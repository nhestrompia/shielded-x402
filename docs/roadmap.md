# Roadmap

This repository now tracks the breaking multi-chain credit architecture:

- single Postgres-backed sequencer
- per-chain relayers (`/v1/relay/pay`)
- delayed Base commitments for auditability
- Solana devnet gateway integration (Noir + Sunspot verifier + native gateway program)

Canonical protocol spec: [`docs/multi-chain-credit-mvp.md`](./multi-chain-credit-mvp.md)
